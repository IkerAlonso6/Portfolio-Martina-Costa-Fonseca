2. CSS/BASE.CSS (Variables y Reset)
Este archivo es el corazón del diseño. Define todas las variables que usas en todo el sitio.
css:root {
    /* Colores Light (modo claro) */
    --color-primary: #8B2C3F;      /* Bordo - color principal */
    --color-secondary: #F5F5F5;    /* Gris clarito - fondos suave */
    --color-text: #2C2C2C;         /* Gris oscuro - texto principal */
    --color-text-light: #666666;   /* Gris medio - texto secundario */
    --color-border: #E8E8E8;       /* Gris muy clarito - líneas */
    --color-bg: #FFFFFF;           /* Blanco puro - fondo */
¿Qué es :root?

Es el selector de HTML raíz
Las variables acá son globales en todo el sitio
Se acceden con var(--color-primary) en cualquier lugar

Ejemplo práctico:

Si Martina quiere cambiar bordo de #8B2C3F a otro color, lo cambias UNA sola vez acá
Automáticamente se actualiza en TODOS lados

css    /* Typography (Tipografía) */
    --font-sans: 'Inter', sans-serif;      /* Letra para texto normal */
    --font-display: 'Montserrat', sans-serif; /* Letra para títulos */
    
    --text-h1: 3rem;      /* 48px - título principal */
    --text-h2: 2rem;      /* 32px - títulos secundarios */
    --text-h3: 1.5rem;    /* 24px - títulos pequeños */
    --text-body: 1rem;    /* 16px - párrafos normales */
    --text-small: 0.875rem; /* 14px - texto pequeño */
¿Por qué variables de tipografía?

3rem = 48px (pero responsivo, se adapta a pantallas)
clamp() (que usamos después) ajusta automáticamente en móvil

css    /* Spacing (Espaciado) */
    --space-xs: 0.5rem;   /* 8px - gaps pequeños */
    --space-sm: 1rem;     /* 16px - espaciado normal */
    --space-md: 1.5rem;   /* 24px - espaciado medio */
    --space-lg: 2rem;     /* 32px - espaciado grande */
    --space-xl: 3rem;     /* 48px - espaciado extra grande */
Para qué sirve?

En lugar de escribir margin: 24px, escribis margin: var(--space-md)
Consistencia: Todos los espacios usan la misma escala
Cambios rápidos: Si querés todo más junito/separado, cambias los valores acá

css    /* Transitions (Animaciones) */
    --transition: 300ms cubic-bezier(0.4, 0, 0.2, 1);
    --transition-slow: 500ms cubic-bezier(0.4, 0, 0.2, 1);
}
Explicación:

300ms = duración (0.3 segundos)
cubic-bezier(0.4, 0, 0.2, 1) = velocidad de la animación (curva suave)
--transition-slow para transiciones más lentas (ej: cambio de página)

css/* Dark mode */
[data-theme="dark"] {
    --color-bg: #0F0F0F;      /* Casi negro */
    --color-text: #EEEEEE;    /* Blanco puro (mejor contraste) */
    --color-text-light: #999999; /* Gris medio */
    --color-border: #2A2A2A;  /* Gris muy oscuro */
}
¿Cómo funciona?

Cuando HTML tiene <html data-theme="dark">...
Estas variables sobreescriben las del :root
Con un simple toggle, todo cambia de color

3. CSS/ANIMATIONS.CSS (Movimiento)
Este archivo define cómo se mueven las cosas.
css@keyframes fadeIn {
    from {
        opacity: 0;   /* Empieza invisible */
    }
    to {
        opacity: 1;   /* Termina visible */
    }
}
¿Qué es @keyframes?

Define pasos de una animación
from = estado inicial
to = estado final
El navegador anima automáticamente entre medio

css@keyframes slideInUp {
    from {
        opacity: 0;
        transform: translateY(30px); /* Empieza 30px más abajo */
    }
    to {
        opacity: 1;
        transform: translateY(0);    /* Termina en posición normal */
    }
}
Efecto visual:

El texto sube suavemente mientras aparece
Eso se ve más profesional que aparecer de golpe

css@keyframes projectFade {
    from {
        opacity: 0;
        transform: translateX(20px);  /* Empieza 20px a la derecha */
    }
    to {
        opacity: 1;
        transform: translateX(0);     /* Llega a posición normal */
    }
}
Para qué: Cuando navegas entre proyectos, entran desde la derecha
css@keyframes pageExit {
    from {
        opacity: 1;  /* Empieza visible */
    }
    to {
        opacity: 0;  /* Termina invisible */
    }
}
Ahora aplicamos estas animaciones:
css.fade-in {
    animation: fadeIn var(--transition) ease-out;
}
/* Significa: 
   - Usa la animación "fadeIn"
   - Dura 300ms (var(--transition))
   - ease-out = empieza rápido, termina lento
*/

.slide-in-up {
    animation: slideInUp var(--transition-slow) ease-out;
}

.project-enter {
    animation: projectFade var(--transition) ease-out;
}

.page-exit {
    animation: pageExit var(--transition) ease-out;
}
Uso en HTML:
html<div class="page fade-in">Esto aparece con fade</div>
Cuando el JS agregue la clase fade-in, automáticamente anima.
css.btn {
    transition: all var(--transition);
    /* "Anima TODOS los cambios de CSS en 300ms" */
}

.btn:hover {
    transform: translateY(-2px);  /* Sube 2px */
    filter: brightness(0.95);     /* Se oscurece ligeramente */
}
Efecto: Al pasar el mouse sobre botón, sube suavemente y se oscurece
